# int128

`usub::umath::int128` is a portable signed 128-bit integer.

## Storage
- Two's complement stored as `(hi, lo)` where `hi` is sign-extended.

## Operations
- `+ - * / %`
- bitwise, shifts
- unary `-`
- comparisons and streaming

## Formatting
- `to_string()` outputs base-10, includes `-` for negative values.
